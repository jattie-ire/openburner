# Openburner

Openburner is a truely opensource stealthburner MCU board based on a raspberry Pi 2040. 

Opensourcemeaning the KiCad files are here, the PCB design is here and you can use the files and modify them and get PCB's made at cost. A toolhead PCB should NOT cost €100, not even €50, A PI2040 and TMC driver is a few cents.

This scenario gets worse of you want to build a Stealthchanger and every head is €100-€200. A decent stealhburner head could cost €35, at least this is what I am aiming for.
